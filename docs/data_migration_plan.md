# User Data Migration Plan

## Backup Paths (Windows)
- C:\Users\<User>\Documents
- C:\Users\<User>\Desktop
- C:\Users\<User>\Pictures
- C:\Users\<User>\Downloads
- Browser bookmarks (Edge/Chrome/Firefox)
- Outlook PST files
- VSCode settings (if applicable)

## Restore Targets (Linux)
- ~/Documents
- ~/Desktop
- ~/Pictures
- ~/Downloads
- ~/.config/<appname>
- ~/Mail / Thunderbird profile
